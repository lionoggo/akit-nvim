# The Ultimate vimrc

This configuration is the ultimate vimrc for neovim.

## How to install ?
### Install for your own user only

```shell
git clone --depth=1 https://github.com/lionoggo/akit-nvim.git ~/.nvim_runtime
sh ~/.nvim_runtime/install_akit_nvim.sh
```
